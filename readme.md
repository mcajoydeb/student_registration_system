Student Registration System

A simple **Student Registration System** built using **HTML, CSS, and Vanilla JavaScript**.  
This application allows users to **add, edit, delete, and persist student data** using **Local Storage**.

---

** Features

- Add new student records
- Edit student details using the same form
- Delete student records
- Input validations:
  - Student Name → Characters only
  - Student ID → Numbers only
  - Email → Valid email format
  - Contact Number → Exactly 10 digits
- Data persistence using **Local Storage**
- Responsive two-column layout using **Flexbox**
- Smooth UI interactions (fade effects, validation messages)

---

** Technologies Used

- HTML5
- CSS3 (Flexbox, transitions)
- JavaScript (ES6)
- Browser Local Storage
